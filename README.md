# Yuza (User)

https://yuza.herokuapp.com

### Requirements
- Ruby 2.4.1
- Sqlite

### Installation

```bash
  $ ./bin/setup

  Or manually...
    $ bundle install
    $ ./bin/rails db:setup
```

### Running
`$ rails server`


### Running the tests
`$ bundle exec rspec`
